## Immediate Rule

For June 3, the deliverable is a proposal, not final experimental results. The deck must describe:

- dataset, collection method, and expected data size
- recommendation problem and significance
- algorithms to implement or apply, with library acknowledgements
- planned experiments
- X-factor

Do not claim that experiments are finished unless the team has real output files and reproducible commands.

## Public Repo Boundary

Do not upload the official school PDF/course document to this public GitHub repository because the local copy is marked restricted. This folder only includes a student-authored summary of the announcement text provided by the team.
